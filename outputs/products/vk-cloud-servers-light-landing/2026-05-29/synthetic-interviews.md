# Синтетические интервью (Synthetic Interviews)

## Guardrail

Синтетические интервью не являются свидетельством реального поведения пользователей. Они используются исключительно для генерации гипотез, исследования лексики целевой аудитории, выявления пограничных случаев и подготовки гайдов для реальных интервью.

## Inputs Used

- `research-summary.md`
- `proto-personas.md`

## Interview Metadata

- **Цель:** Стресс-тестирование возражений покупателей и подготовка гайда реального интервью.
- **Персоны / Сегменты:** Александр (CTO / основной покупатель), Дмитрий (Финдир / B2B-закупщик), сомневающийся посетитель.
- **Версия скрипта:** Сгенерировано в рамках этапа исследований воркфлоу.
- **Статус доказательств:** `synthetic` (синтетический)
