# Run Plan

## Статус

`success`

## Запрос

Сделать сайт по продаже Valorant Points. Пользователь дал референс: `https://pixelperfect.school/ai-native-designer`.

## План этапов

- 00-intake: recursive brief, handoff, ledger.
- 01-research: deep research по Riot legal/IP, prepaid card support, аудитории, JTBD, конкуренты, риски.
- 02-prd: требования к безопасному неофициальному marketplace лендингу.
- 03-ia: структура страницы и главный сценарий.
- 04-design: оригинальная editorial commerce визуальная система по мотивам sales-ритма референса без прямого копирования.
- 05-copy: disclaimer, CTA, карточки пакетов, FAQ.
- 06-screens: спецификация экранов.
- 07-prototype: transition map.
- 08-frontend: реализация.
- 09-visual-reference-review: full-page screenshot-сверка всего сайта с референсом.
- 10-test-bench: funnel analytics.
- 11-qa: проверка.
- 12-release: notes.

## Ограничения

- Полное копирование дизайна Riot/VALORANT или Pixel Perfect, логотипов, UI, trade dress, текстов и ассетов не выполняется.
- Из референса разрешены только общие паттерны: крупная типографика, нумерованные секции, плотные карточки и заметные CTA.
- Сайт должен явно обозначать неофициальный статус.
- Нельзя просить Riot login/password.
- Региональная совместимость prepaid/cash codes должна быть заметной.
- Notion publication не выполняется без target page/database и approval.
