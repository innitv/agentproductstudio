## Visual Direction
Наш дизайн-концепт — **Dark Tech Premium**. Мы используем глубокие оттенки серого и сланцевого цветов для фона, яркие контрастные градиенты для акцентов и закругления 12px для придания интерфейсу современного, дорогого вида.

*   **Цветовая палитра (Figma A3 Tokens)**:
    *   Основной фон панели: `var(--neutral-990)` (#0F1D2D / темно-синий глубокий)
    *   Фон карточек и сайдбара: `var(--neutral-950)` (#191C20 / антрацит)
    *   Акцентный синий (кнопки, ссылки): `var(--base-info)` (#1998FF)
    *   Акцентный зеленый (активный статус): `var(--base-success)` (#2BB673)
*   **Типографика**: Использование шрифта `Mont` для заголовков (размер h1: `32px`, h2: `24px`) и `Inter` для текста таблиц и сообщений чата (размер: `14px`, `13px`).

## Sections
- **Sidebar Navigation**: Панель навигации по разделам платформы.
- **Analytics Dashboard**: Карточки метрик и аналитика эффективности.
- **AI Agents Table Grid**: Сводная таблица агентов со статусами.
- **Chat Simulator Widget**: Интерактивная песочница диалогов с агентом.
- **Create Agent Form**: No-code конструктор с выбором модели и регламента.

## Components
Мы интегрируем **9 кастомных UI-компонентов** нашей дизайн-системы:
1.  `Sidebar`: левая навигация с иконками.
2.  `MetricCard`: карточки аналитики.
3.  `Button`: основные и второстепенные кнопки.
4.  `SwitchToggle`: switch-переключатели статуса агентов в таблице.
5.  `ChatWindow`: контейнер переписки в симуляторе.
6.  `InputField` / `SelectMenu` / `TextArea`: контролы формы создания агента.
7.  `ToastNotification`: всплывающие уведомления при успешных действиях.

## Responsive Notes
*   **Mobile view**: Сайдбар на мобильных экранах переходит во вспомогательную нижнюю панель или меню-гамбургер. Все блоки (аналитика, таблица, чат) выстраиваются вертикально в одну колонку.
*   **Desktop view**: Классический двухколоночный макет с фиксированным левым сайдбаром (280px) и гибкой правой зоной.

## Accessibility Notes
- Использование семантических тегов HTML5 (main, header, footer, aside, nav).
- Наличие явных меток `aria-label` для инпутов, кнопок переписки и иконок удаления.
- Поддержка навигации с клавиатуры и явные фокус-состояния для всех интерактивных кнопок A3.
