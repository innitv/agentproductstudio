## Disallowed Copying

| Element | Why disallowed | Replacement direction |
|---|---|---|
| Riot/VALORANT logos | trademark/affiliation risk | Text disclaimer and VP Nexus brand |
| Official agent/key art/maps/screenshots | copyrighted game assets | Generated abstract prepaid code cards |
| Official site layout/trade dress | lookalike confusion risk | Оригинальный порядок секций и component styling |
| Official typography/color system as exact clone | brand identity copying | Custom graphite/red-orange/cyan palette |

## Design Implications

- Layout: region-first marketplace, not official game marketing homepage.
- Color: graphite base with red-orange/cyan accents, not exact VALORANT site palette.
- Typography: system font, not official brand typography.
- Imagery: generated abstract code-card render, no Riot assets.
- Interaction: package selection and region chips, not official game content navigation.
- Accessibility: maintain contrast and native controls.

## Decision

Use as inspiration only. Do not copy official design. Build an original independent marketplace with explicit disclaimer.
