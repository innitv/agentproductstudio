# Recursive Brief

## Inputs Used

- User request

## Expansion

Пользователь хочет сайт для продажи Valorant Points. Запрос включает требование полностью копировать дизайн сайта VALORANT, но это ограничено правами Riot и риском копирования trade dress.

## Deepening

Нужно сделать коммерческий лендинг, который объясняет покупку prepaid/cash codes, помогает выбрать регион, подчеркивает безопасность аккаунта и не просит Riot credentials. Визуально сайт может быть gaming/tactical, но должен быть оригинальным.

## Consolidation

Создать лендинг `VP Nexus`: независимый marketplace концепт для region-ready prepaid/cash codes. Основной сценарий: выбрать регион -> посмотреть пакеты -> проверить правила безопасности -> перейти к checkout placeholder. Не использовать Riot assets/logos/UI copy.

## Assumptions

- Это независимый магазин, не Riot.
- Продаются коды, а не аккаунт-сервис.
- Пользователь хочет коммерческую страницу, значит фан-политика Riot не применима как разрешение на использование IP.
