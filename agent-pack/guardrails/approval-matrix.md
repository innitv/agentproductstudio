# Approval Matrix

| Action | Approval |
|---|---|
| Notion publish/update | required |
| External research API call with project/product context | required unless user already approved this provider for the run |
| Required provider unavailable or approval denied | blocked/partial, must be recorded |
| Deploy | required |
| Delete data | required |
| Change secrets | required |
| Send external messages | required |
| Local artifact write | not required |
| Local validation command | not required |
