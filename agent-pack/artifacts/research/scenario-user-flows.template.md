## Зачем нужен этот слой

Этот артефакт переводит выводы исследования, матрицы и CJM в реальные пользовательские маршруты. Пользователь не взаимодействует с абстрактной возможностью или способом оплаты: он приходит в ситуацию, задает вопрос, делает действие, ждет статус, сталкивается с исключением и должен понять доказательство результата.

Минимальная проверка качества: персона, ситуация, шаги, статус, исключение, доказательство и способ проверки должны быть видны в каждом P0/P1 флоу.

## Проверка флоу

| Проверка | Как понять, что флоу готов | Evidence status |
|---|---|---|
| Пользователь может пересказать путь | Он объясняет, что происходит до действия, во время ожидания, после результата и при исключении |  |
| Статусы не смешиваются | Принято, ожидается, подтверждено, отменено и возвращено не называются одним словом |  |
| Есть доказательство результата | Для каждого финала указан документ, событие, источник статуса, чек, ссылка, акт или другой проверяемый след |  |
