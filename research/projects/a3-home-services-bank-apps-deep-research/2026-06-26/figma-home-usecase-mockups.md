## Visual Evidence Notes

- Основная визуальная правда: предоставленные пользователем экраны Альфа-Банка.
- Перенесенные паттерны: белый фон, крупные светло-серые cards, черная primary CTA, адресный black pill, bottom nav, короткие подписи, минимум обучающего текста.
- Логотипы поставщиков не копировались: вместо них использованы нейтральные цветовые marks, чтобы не имитировать чужие assets.
- Шрифт: использован `Inter` как доступный fallback для iOS-like системной типографики. При наличии точного Alfa/SF Pro font stack в Figma файл можно заменить.

## Lazyweb Gate

Lazyweb был вызван как обязательный UI evidence layer, но backend вернул hard stop: установленный skill-pack устарел и требует `0.13.0+`.

Прямой `curl -fsSL https://www.lazyweb.com/install.sh | bash` был отклонен auto-review как непиннутый внешний install script с записью вне workspace. По согласованию с пользователем макеты собраны по второму маршруту: без Lazyweb, с фиксацией blocker.
